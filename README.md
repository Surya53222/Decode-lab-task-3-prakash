# Decode-lab-task-3-prakash
Database Integration

📌 Project Overview

This project focuses on integrating a database with a backend application to store, retrieve, update, and manage data efficiently.

The application demonstrates how backend services communicate with databases using structured queries and data models.

---

🎯 Objectives

- Connect application with a database.
- Perform CRUD operations.
- Store and retrieve data efficiently.
- Manage persistent application data.

---

🚀 Features

- Database Connectivity
- Create Records
- Read Records
- Update Records
- Delete Records
- Data Validation
- Persistent Storage

---

🛠 Technologies Used

Backend

- Node.js
- Express.js

Database

- MongoDB / MySQL

- MongoDB Compass / MySQL Workbench
